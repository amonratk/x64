# How to enable some extension

## enabled mbstring

1. Checking exist dll first `C:\tools\php74\ext` should be exist `php_mbstring.dll`
2. Enabled via uncommend file on `C:\tools\php74\php.ini`

```bash
;extension=mbstring
extension=mbstring
```
