# How to create local domain

## HOSTS

Go to `C:\Windows\System32\drivers\etc\hosts`

```bash
# localhost name resolution is handled within DNS itself.
#	127.0.0.1       localhost
#	::1             localhost
127.0.0.1           dashboard.local
127.0.0.1           yii2.local
```
