# How to installation CLI

## Download

Go to `https://symfony.com/download` or <a href="https://github.com/symfony/cli/releases/download/v4.12.8/symfony_windows_amd64.exe">Click(64bit)</a>

## Install

Double click `symfony_windows_amd64.exe`

> symfony -V
